# Planetary Gear Mechanism

## Overview
Design and motion simulation of a planetary gear mechanism created in PTC Creo Parametric and Creo Mechanism as part of the CodeAlpha Mechanical Design Internship (Task 3).

## Tools Used
- PTC Creo Parametric
- Creo Mechanism

## Contents
- `design_files/` — Creo part and assembly files (.prt, .asm)
- `simulation/` — Motion simulation results (angular position, velocity, acceleration)
- `images/` — Renders and screenshots
- `report/` — Task report (PDF/Word)

## Description
The planetary gear mechanism (sun gear, planet gears, ring gear, and carrier) was assembled in Creo, then a motion simulation was run in Creo Mechanism to analyze angular position, velocity, and acceleration of the gear elements.
